# Frankfurter

[![Build](https://github.com/lineofflight/frankfurter/workflows/build/badge.svg)](https://github.com/lineofflight/frankfurter/actions)

[Frankfurter](https://frankfurter.dev) is a free and open-source currency data API that tracks reference exchange rates published by institutional and non-commercial sources like the European Central Bank. Check the website for a detailed walkthrough.

The API is publicly available at <https://api.frankfurter.dev>.

## Deployment

### Using Docker

The simplest way to run Frankfurter locally is with Docker:

```bash
docker build -t frankfurter:local .

docker run -d -p 8000:8000 lineofflight/frankfurter
```

Once the container is running, open your browser and go to `http://localhost:8000`.

## FX Summary API

The service provides endpoints for analyzing foreign exchange rate data over date ranges.

### Health Check

Check if the service is running:

```bash
curl http://localhost:8000/health
```

Response:
```json
{
  "status": "ok",
  "timestamp": "2026-02-05T12:00:00Z"
}
```

### Summary Endpoint

Get a summary of exchange rates for a date range:

```bash
curl "http://localhost:8000/summary?start_date=2025-07-01&end_date=2025-07-03&from=EUR&to=USD"
```

With daily breakdown:

```bash
curl "http://localhost:8000/summary?start_date=2025-07-01&end_date=2025-07-03&from=EUR&to=USD&breakdown=day"
```

Response:
```json
{
  "base": "EUR",
  "start_date": "2025-07-01",
  "end_date": "2025-07-03",
  "totals": {
    "start_rate": 1.182,
    "end_rate": 1.179,
    "total_pct_change": -0.2538,
    "mean_rate": 1.182
  },
  "breakdown": [
    {
      "date": "2025-07-01",
      "rate": 1.182,
      "pct_change": null
    },
    {
      "date": "2025-07-02",
      "rate": 1.185,
      "pct_change": 0.2538
    },
    {
      "date": "2025-07-03",
      "rate": 1.179,
      "pct_change": -0.5063
    }
  ]
}
```

### Web UI

Access the interactive web interface at:

```
http://localhost:8000/ui
```

The UI allows you to:
- Select start and end dates
- Choose currency pairs (from/to)
- View summary totals and daily breakdown
- See percentage changes with color coding

### Features

- **API Integration**: Fetches data from the Frankfurter public API
- **Fallback**: Automatically falls back to local `data/sample_fx.json` if the network fails
- **Retry Logic**: Automatically retries failed API requests (up to 3 times)
- **Caching**: In-memory cache with 5-minute TTL to reduce API calls
- **Error Handling**: Graceful handling of division by zero and missing data

## Libraries / Tools

- [beancount-exchangerates](https://github.com/xuhcc/beancount-exchangerates): a [Beancount](https://beancount.github.io/) price source plugin using this API
- [frankfurte-rs](https://github.com/Rolv-Apneseth/frankfurte-rs): Rust library and CLI to interface with this API

## Contributing

Frankfurter is built with Ruby. To contribute:

1. Fork.
2. Install dependencies with `bundle install`.
3. Run tests with `bundle exec rake`.
4. Push your changes to a feature branch.
5. Open a pull request.

### Reporting Missing Currencies

If you notice a currency missing from our data, please open an issue and include a suggested source with the missing data. We're looking for non-commercial sources like the European Central Bank that publish current and historical daily rates at the end of each working day.
